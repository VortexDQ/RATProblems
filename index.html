Complete Guide: Remove Viruses & Malware

Follow these steps carefully to clean your computer

Important: Do not skip any steps. Following these instructions in order is crucial for successfully removing malware from your system.

1

Disconnect from WiFi/Internet

Disconnecting from the internet prevents the malware from communicating with external servers, downloading additional threats, or spreading to other devices on your network.

How to Disconnect from WiFi:

1

Click the Network icon in the system tray

Located in the bottom-right corner of your screen (next to the clock)

2

Click on your connected WiFi network

It will show as "Connected"

3

Click "Disconnect"

Your WiFi icon should now show as disconnected

Alternative Method:

Turn off your WiFi adapter using the physical switch on your laptop, or unplug your ethernet cable if using a wired connection.

2

Boot into Windows Safe Mode

Safe Mode starts Windows with only the essential drivers and services, preventing malware from loading automatically. This makes it easier to detect and remove threats.

Method 1: Using System Configuration (Recommended)

1

Press Windows key + R

This opens the Run dialog box

2

Type "msconfig" and press Enter

The System Configuration window will open

3

Click on the "Boot" tab

This tab contains boot configuration options

4

Under "Boot options", check the "Safe boot" checkbox

Select "Minimal" for the best results

5

Click "Apply" and then "OK"

You'll be prompted to restart

6

Click "Restart"

Your computer will reboot into Safe Mode

Method 2: Using Settings (Windows 10/11)

1

Hold Shift and click Restart from the Start menu

2

Select Troubleshoot > Advanced options > Startup Settings > Restart

3

Press F4 to enable Safe Mode

Remember:

After cleaning your computer, you'll need to disable Safe Mode by going back to msconfig, unchecking "Safe boot", and restarting normally.

3

If You Already Have Antivirus:

Open your antivirus program

Update virus definitions to the latest version

Run a Full System Scan (not quick scan)

Wait for the scan to complete (may take 1-3 hours)

Quarantine or delete all detected threats

4

Delete Temporary Files

Malware often hides in temporary files. Clearing these can remove residual threats and free up disk space.

1

Press Windows key + R

Opens the Run dialog

2

Type "temp" and press Enter

This opens your temporary files folder

3

Select all files (Ctrl + A) and delete them

Some files may be in use and can't be deleted - that's okay, skip them

4

Repeat with "%temp%" and "prefetch"

Type each in the Run dialog to access more temp folders

5

Check Browser Extensions

Malicious browser extensions can hijack your browsing, steal data, or display unwanted ads.

For Google Chrome:

  1. Type chrome://extensions in the address bar
  2. Review all installed extensions
  3. Remove any extensions you don't recognize or didn't install
  4. Disable suspicious extensions even if you're not sure

For Microsoft Edge:

  1. Type edge://extensions in the address bar
  2. Review and remove unwanted extensions

For Firefox:

  1. Click the menu button (three lines) → Add-ons and themes
  2. Review Extensions tab and remove suspicious ones
6

Restore Normal Boot & Update

Now that you've cleaned your system, it's time to boot back into normal mode and update everything.

Disable Safe Mode:

1

Press Windows key + R, type msconfig

2

Go to the Boot tab

3

Uncheck "Safe boot"

4

Click Apply → OK → Restart

Update Your System:

Go to Settings → Windows Update

Click Check for updates

Install all available updates

Update your antivirus software to the latest version

7

Change Your Passwords

Malware may have captured your passwords. Change them immediately to secure your accounts.

Priority Accounts to Update:

  • • Email accounts
  • • Banking and financial services
  • • Social media accounts
  • • Online shopping accounts
  • • Any account with saved payment information

Password Best Practices:

  • Use passwords with at least 12 characters
  • Include uppercase, lowercase, numbers, and symbols
  • Use unique passwords for each account
  • Consider using a password manager like Bitwarden or LastPass

9

Last Resort: Factory Reset Your Computer

⚠️ Warning: Only Use This as a Last Resort!

If the malware persists after trying all the steps above, or if your system is severely compromised, a factory reset is the nuclear option. This will delete ALL your files, programs, and settings. Only proceed if you've backed up important data to an external drive or cloud storage.

A factory reset will restore your Windows PC to its original state, completely removing all malware, but also everything else.

Before You Begin:

Back up all important files to an external hard drive, USB drive, or cloud storage (Google Drive, OneDrive, etc.)

Write down or save your Windows product key if you don't have a Microsoft account linked

Make a list of programs you need to reinstall and know where to download them

Ensure your laptop is plugged in - never perform a factory reset on battery power

Step-by-Step Factory Reset Guide (Windows 10/11):

1

Open Windows Settings

Press Windows key + I on your keyboard

Or click Start menu → Settings (gear icon)

2

Navigate to Reset Options

Windows 11: Click System → Recovery

Windows 10: Click Update & Security → Recovery

3

Start the Reset Process

Under "Reset this PC", click the Reset PC button

A new window will appear with reset options

4

Choose Your Reset Option

Option 1: Remove Everything (Recommended for Malware)

This completely wipes your drive and reinstalls Windows fresh. Best for removing persistent malware.

⚠️ All files, apps, and settings will be deleted

Option 2: Keep My Files

Removes apps and settings but keeps personal files. Less thorough for malware removal.

⚠️ May not remove all malware if it's embedded in files

💡 For Severe Infections: Choose "Remove Everything" for the most thorough cleaning

5

Choose How to Reinstall Windows

●

Cloud download (Recommended)

Downloads a fresh copy of Windows (requires internet, ~4GB download)

●

Local reinstall

Uses Windows files already on your computer (faster, no download needed)

6

Additional Settings (Optional but Recommended)

Click "Change settings" to access advanced options:

Clean data fully? Yes (Recommended)

Securely erases files so they can't be recovered. Takes longer but more secure.

Delete files from all drives? Your choice

If malware spread to other drives, enable this. Otherwise, just clean the Windows drive.

7

Review and Confirm

Windows will show you a summary of what will happen. Read it carefully!

When you're ready, click Reset

8

Wait for the Process to Complete

Your computer will restart several times

The process can take 30 minutes to 2 hours depending on your hardware

Don't turn off your computer during this process!

☕ Grab a coffee and let Windows do its thing. Progress will be shown on screen.

9

Set Up Windows Again

Once complete, you'll go through the initial Windows setup like a brand new computer:

  • Choose your language, region, and keyboard layout
  • Sign in with your Microsoft account
  • Configure privacy settings
  • Install Windows updates immediately
  • Install antivirus software before doing anything else
  • Reinstall your programs one by one
  • Scan your backed-up files with antivirus before restoring them

Congratulations! Your Computer is Now Clean

You now have a fresh, malware-free Windows installation. Follow the prevention tips below to keep it that way, and always practice safe browsing habits moving forward!

Prevent Future Infections

Keep Software Updated

Regularly update Windows, browsers, and all installed programs. Security patches fix vulnerabilities that malware exploits.

Be Cautious with Downloads

Only download software from official websites. Avoid pirated software, cracks, and "free" versions of paid programs.

Think Before Clicking

Don't click suspicious email links or attachments. Verify sender identity before opening anything unexpected.

Use Real-Time Protection

Keep your antivirus active with real-time scanning enabled. Schedule regular automatic scans.

Enable Firewall

Windows Firewall should always be enabled. It blocks unauthorized network connections to your computer.

Create Backups

Regularly back up important files to an external drive or cloud storage. This protects against ransomware.
